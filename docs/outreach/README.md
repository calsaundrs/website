## What to expect

Backlinks are slow signals. A successful outreach lands a link in the next 1–4 weeks. The first one is the hardest; subsequent links get easier because of the editorial precedent. Track in GSC's "Links" report and the monthly SEO automation.

## Etiquette notes

- **Lead with usefulness**, not the ask. Each draft offers something to the recipient before asking for anything.
- **Don't lead with SEO talk.** No one cares about your domain authority. Talk about the readers / customers / community.
- **One link, one ask.** Don't bundle multiple requests into a single message.
- **No follow-ups within 5 days.** Soft nudge after a week is fine; multiple chases look desperate.
