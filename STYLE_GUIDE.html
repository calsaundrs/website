MANIFESTO

The visual rulebook for Brum Outloud. Heavy borders, toxic shadows, and unapologetic typography.

01. Colors & Usage

Our palette is dark, moody, and highly contrasted. We use neon accents strictly for interactive elements, highlights, and hard borders.

Deep Void

#0D0115

Primary background color. A very dark, slightly purple-tinted black.

DO: Use for main body background.

DO: Use true black (#000) for nested cards.

Toxic Lime

#CCFF00

Primary action color. The highest contrast accent in the palette.

DO: Use for primary buttons and CTAs.

DO: Use for prominent text highlights.

DON'T: Use as a background for large text.

Hot Pink

#E83A99

Secondary accent. Used for hover states, secondary borders, and warnings.

DO: Use for hover borders on cards.

DO: Use for secondary action stickers.

DON'T: Use next to Toxic Lime without a dark border separating them.

Electric Purple

#9B5DE5

Tertiary accent. Used for subtle drop shadows, categorization, and backgrounds.

DO: Use for solid brutalist drop shadows.

DO: Mix with 10% opacity for section backgrounds.

Light Text

#F3E8FF

A slightly purple-tinted off-white to reduce eye strain against the stark black.

DO: Use for all standard headers and body text.

DO: Reduce to text-gray-200 or text-gray-300 for hierarchy.

Progress Pride Gradient

Used specifically for footer boundaries and the main top nav boundary. Class: .progress-pride-bg

02. Typography Rules

Our typography is loud and unabashed. We use Syne for display and impact, and Space Grotesk for technical precision and readability.

DISPLAY FONT

Syne

Used exclusively for major headings, hero text, and high-impact branding. It must always be uppercase and always font-black (800).

DO:

  • Use for H1, H2, and brand headers.
  • Always set text-transform to uppercase.
  • Apply .misprint for text 5xl and above to tighten tracking and add text shadows.

DON'T:

  • Never use Syne for paragraph text or long sentences.
  • Never use lowercase letters.

BODY & UI FONT

Space Grotesk

Used for everything else: paragraphs, buttons, forms, labels, and metadata. It brings a technical, structured balance to the chaotic display font.

DO:

  • Use for all descriptive text (text-base to text-lg).
  • Use heavy weights (bold) with uppercase and tracking (tracking-widest) for small labels and buttons.
  • Set body line-height to 1.6 (relaxed) to contrast the tight headers.

WATERMARK FONT

Bungee Outline

Used specifically for massive, viewport-spanning background watermarks. This font has aggressive impact but with zero visual weight thanks to the thin outline construction.

DO:

  • Use via the .watermark-bg component or .font-accent utility.
  • Render at absolutely massive scale (e.g. 25vw+).
  • Set opacity extremely low (10-20%) so it sits deep in the background.
