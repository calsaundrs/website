Opening Status System

Venue Opening Status States

The opening status system automatically calculates and displays the current status of venues based on their opening hours and the current time.

Open Status

Open

Open until 2:00 AM

Color: Green (bg-green-500/10 text-green-400 border-green-500/30)

Trigger: Current time is within opening hours

Message: Shows closing time

Closed Status

Closed

Currently Closed

Color: Red (bg-red-500/10 text-red-400 border-red-500/30)

Trigger: Current time is outside opening hours

Message: "Currently Closed"

Opens Soon Status

Opens Soon

Opens at 8:00 PM

Color: Orange (bg-orange-500/10 text-orange-400 border-orange-500/30)

Trigger: Venue opens within 60 minutes

Message: Shows opening time

Implementation Notes

Status Calculation Logic

  • Uses London timezone (Europe/London)
  • Handles overnight hours (e.g., 11 PM - 2 AM)
  • Considers "Opens Soon" when venue opens within 60 minutes
  • Supports multiple time slots per day
  • Handles "Closed" days and "Not Available" data

Information Layout

  • • Enhanced about section with detailed description

  • • Venue type tags with hover effects
  • • Location and website info cards

Visual Enhancements

  • • Gradient backgrounds with purple/blue themes
  • • Glassmorphism cards with backdrop blur
  • • Hover animations and transitions
  • • Icon integration throughout
  • • Responsive design patterns

Events Section

  • • Separated recurring vs one-off events
